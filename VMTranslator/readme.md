## Code à écrire en VM dans des fichiers.vm dans le dossier VMTranslator/test

---------------

Chaque fichier peut contenir une *ou plusieurs* fonction(s), chaque fonction doit commencer par :

__Function nomFonction numArgsLocaux__

---------------

Merci d'utiliser **Main.vm**, pour l'utilisation des fonctions et appels possibles se référer aux commentaires des fonctiones

Exemple : 
>def fonction(self, command):
