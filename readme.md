## **Règles du jeu "Circle Die"**

### **Objectif :**  
Trouvez et attrapez **le cercle gagnant** pour remporter la partie !  

### **Commandes :**  
- Déplacez votre personnage (**le carré**) avec les **flèches directionnelles** du clavier.  
- Ajustez la taille de votre personnage :  
  - Appuyez sur **A** pour **agrandir**.  
  - Appuyez sur **R** pour **réduire**.  

### **Attention :**  
- Certains cercles peuvent vous faire **perdre**.  
- Observez bien et planifiez votre approche pour trouver le **cercle gagnant** et la manière de l'atteindre.

## **Description du projet**  

Nous avons initialement tenté de recréer le jeu **"The Hardest Game"**. Cependant, après plusieurs essais, nous avons constaté que l’utilisation de l’aléatoire était trop complexe à gérer en raison des contraintes de mémoire.  

En nous basant sur un projet existant (un jeu de labyrinthe) étudié dans le cours, nous avons tenté d’adapter ce concept. Malheureusement, cette approche s’est avérée trop difficile à mettre en œuvre dans le cadre de ce TD.  

Nous avons finalement opté pour une base plus accessible : le **"Square Game"**, disponible sur le site de **Nand2Tetris**. En partant de ce modèle, nous avons développé notre propre jeu, intitulé **Circle Die**.
